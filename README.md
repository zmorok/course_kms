# Курсовой проект по КМС: промышленный резак для бумаги

Репозиторий содержит сайт курсового проекта по дисциплине **«Компьютерные мультимедийные системы»** на тему **«Промышленный резак для бумаги»**.

Сайт предназначен для изучения устройства, назначения и принципа работы промышленного гильотинного резака для бумаги. В проекте объединены теоретические материалы, визуальное описание оборудования, анимация рабочего цикла, встроенный симулятор и диалоговая база знаний.

## Что реализовано на сайте

- **Титульная страница** с информацией о курсовом проекте, дисциплине, теме и авторе.
- **Теоретический раздел** об имитационном моделировании, его методах, принципах и применении для обучения работе с техническими установками.
- **Описание промышленного резака для бумаги**: назначение, история, области применения, основные узлы станка, материалы, ограничения, дефекты реза и требования безопасности.
- **Интерактивное изучение устройства**: на странице описания есть схема с зонами промышленного резака, по которым можно получить информацию об отдельных элементах.
- **Принцип работы в виде анимации**: отдельная страница показывает рабочий цикл резака через видео-анимацию и пошаговое описание процесса.
- **Симулятор промышленного резака**: на сайт встроена Unity WebGL-сборка симулятора. Исходный проект симулятора находится в отдельном репозитории: [course_unity](https://github.com/zmorok/course_unity).
- **База знаний в виде диалога**: пользователь может вводить вопросы по теме промышленного резака и получать ответы на основе подготовленной базы знаний.

## Структура проекта

```text
.
├── index.html                 # Титульная страница
├── pages/                     # Основные страницы сайта
│   ├── theory.html            # Теория по имитационному моделированию
│   ├── description.html       # Описание промышленного резака
│   ├── work-algorithm.html    # Принцип работы и анимация
│   ├── simulator.html         # Страница со встроенным Unity-симулятором
│   └── dialog.html            # Диалоговая база знаний
├── assets/                    # Стили, скрипты, изображения, видео и аудио
└── vendor/unity/              # WebGL-сборка Unity-симулятора
```

## Запуск

Проект является статическим сайтом. Для просмотра можно открыть файл `index.html` в браузере.

Если браузер блокирует локальную загрузку Unity WebGL-сборки, сайт лучше запускать через локальный HTTP-сервер, например через команду ниже либо через расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer):

```bash
python -m http.server 8000
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:8000
```
